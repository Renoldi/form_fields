# AppFabMenu

Expandable floating action button menu for grouped quick actions.

## Basic Usage

```dart
AppFabMenu(
  items: [
    AppFabMenuItem(label: 'First', icon: Icon(Icons.looks_one), onPressed: () {}),
    AppFabMenuItem(label: 'Second', icon: Icon(Icons.looks_two), onPressed: () {}),
  ],
)
```
