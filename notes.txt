Prompt profesional untuk membuat contoh halaman baru dengan struktur PresenterState/View (seperti di settings):

Buatkan contoh halaman Flutter dengan struktur berikut:
- Gunakan pola Presenter-View-ViewModel seperti di halaman settings.
- Buat file terpisah: main.dart, presenter.dart, view.dart, view_model.dart.
- Kelas Presenter extends StatefulWidget.
- Kelas PresenterState extends State<Presenter> (abstract).
- Kelas View extends PresenterState, berisi UI utama.
- Kelas ViewModel untuk state dan logic.
- Sertakan minimal satu interaksi sederhana (misal tombol, input, dsb).

Contoh: halaman "ContohCounter" dengan tombol tambah dan tampilkan nilai counter, seluruh logic dan state dikelola di ViewModel, UI di View, dan struktur mengikuti pola di atas.
